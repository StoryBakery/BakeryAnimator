## 개요

키프레임에 대한 데이터 타입 모음입니다.

## InterpolationMode

키프레임 간의 보간 방식을 정의합니다.

- `"Constant"`: 보간 없이 해당 키프레임 값을 유지합니다 (Step).
- `"Linear"`: 선형 보간을 수행합니다.
- `"Bezier"`: 베지어 곡선 보간을 수행합니다. (핸들 속성 필요)

## HandleType

베지어 핸들의 동작 방식을 정의합니다. (Blender 호환)

- `"Free"`: 핸들을 독립적으로 조절할 수 있습니다 (방향/길이 개별).
- `"Aligned"`: 좌우 핸들의 방향이 일직선으로 유지되지만 길이는 독립적입니다.
- `"Vector"`: 자동 선형 보간을 생성합니다 (핸들 조작 시 Free로 변경).
- `"Automatic"`: 부드러운 곡선을 위해 핸들 위치가 자동 계산됩니다.
- `"AutoClamped"`: 오버슈트(Overshoot)를 방지하며 자동으로 핸들이 계산됩니다.

## KeyframeType

키프레임의 역할이나 상태(일반, 극단값, 중간 과정 등)를 시각적으로 구분하기 위해 정의하는 타입입니다.

`Keyframe` 
일반 - 하얀색
선택시 노란색

일반 키프레임

`Breakdown` 
일반 - 작은 청록색
선택 - 작은 푸른색

주요 포즈 사이의 중간 동작을 나타냅니다, 사이의 전환을 위해 사용됩니다.


`MovingHold` 
일반 - 어두운 회색
선택 - 주황색

겉보기에는 멈춰 있는 것처럼 보이지만,
실제로는 아주 미세하게 계속 움직이고 있는 상태.

도프 시트에서는 그 사이에 막대가 표시됩니다.

`Extreme` 
일반 - 연분홍색
선택 - 진분홍색

움직임의 **끝 지점** 이자 **“에너지의 정점** 을 나타냅니다.

`Jitter` 
일반 - 연한 초록색
선택 - 진한 초록색

Filler 키프레임 (사이를 채우기 위해 자동으로 들어간 키) 이나 Baked 키프레임 (계산으로 베이크된 키), 혹은 그외.

계산 또는 보간 결과로 생성된 "채움용 키"입니다.

`Generated` 
일반 - 회갈색
선택 - 약간 진한 회갈색

다른 툴, 제3 플러그인에 의해 자동생성된 키프레임.
제거하거나 재생성해도 괜찮다는 의미로 사용됩니다.

수동으로 만든 키프레임을 해당 타입으로 설정하지 않게 주의하세요.
